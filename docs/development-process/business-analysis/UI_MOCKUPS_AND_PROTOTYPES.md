# 教育机构通知系统 UI/UX 设计文档

## 📋 设计概述

### 设计理念
基于yudao-boot-mini框架，为教育机构打造专业、易用的多级通知管理系统界面。设计遵循"简洁明了、层次清晰、操作高效"的原则，充分考虑教育行业用户的使用习惯和复杂权限体系的界面呈现需求。

### 核心设计目标
- **角色适配**: 为25+角色提供个性化的界面体验
- **信息清晰**: 四级通知分类的直观展示
- **操作简便**: 复杂工作流的简化交互
- **响应迅速**: 实时数据的高效展示
- **体验一致**: 跨平台的统一视觉语言

```javascript
  filters.value = newFilters

})
```

```javascript
  state: () => ({
    notifications: [],
    loading: false,
    unreadCount: 0
  }),

  actions: {
    async fetchNotifications() {
      this.loading = true
      try {
        const response = await notificationAPI.getList()
        this.notifications = response.data
        this.unreadCount = response.unreadCount
      } catch (error) {
        this.$toast.error('获取通知失败')
      } finally {
        this.loading = false
      }
    },

    async markAsRead(notificationId) {
      try {
        await notificationAPI.markRead(notificationId)
        const notification = this.notifications.find(n => n.id === notificationId)
        if (notification) {
          notification.status = 'read'
          this.unreadCount--
        }
        this.$toast.success('已标记为已读')
      } catch (error) {
        this.$toast.error('操作失败')
      }
    }
  },

  getters: {
    getFilteredNotifications: (state) => (filters) => {
      return state.notifications.filter(notification => {
        if (filters.type !== 'all' && notification.type !== filters.type) {
          return false
        }
        if (filters.status !== 'all' && notification.status !== filters.status) {
          return false
        }
        return true
      })
    }
  }
})
```

### 3. 实时通信

#### WebSocket 集成
```javascript
// composables/useWebSocket.js
import { ref, onMounted, onUnmounted } from 'vue'
import { useNotificationStore } from '@/stores/notification'

export function useWebSocket() {
  const connected = ref(false)
  const notificationStore = useNotificationStore()
  let ws = null

  const connect = () => {
    const token = localStorage.getItem('token')
    ws = new WebSocket(`ws://localhost:8080/ws/notifications?token=${token}`)
    
    ws.onopen = () => {
      connected.value = true
      console.log('WebSocket连接成功')
    }
    
    ws.onmessage = (event) => {
      const data = JSON.parse(event.data)
      handleMessage(data)
    }
    
    ws.onclose = () => {
      connected.value = false
      console.log('WebSocket连接断开')
      // 重连逻辑
      setTimeout(connect, 5000)
    }
    
    ws.onerror = (error) => {
      console.error('WebSocket错误:', error)
    }
  }
  
  const handleMessage = (data) => {
    switch (data.type) {
      case 'NEW_NOTIFICATION':
        notificationStore.addNotification(data.notification)
        showNotification(data.notification)
        break
      case 'STATUS_UPDATE':
        notificationStore.updateNotificationStatus(data.notificationId, data.status)
        break
    }
  }
  
  const showNotification = (notification) => {
    if (Notification.permission === 'granted') {
      new Notification(notification.title, {
        body: notification.content.substring(0, 100),
        icon: '/favicon.ico',
        badge: '/badge.png'
      })
    }
  }
  
  onMounted(() => {
    connect()
  })
  
  onUnmounted(() => {
    if (ws) {
      ws.close()
    }
  })
  
  return {
    connected
  }
}
```
